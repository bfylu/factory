# 简介
工厂(factory)处理创建对象的细节.

定义简单工厂:简单工厂其实不是一个设计模式,反而比较像是一种编程习惯.但由于经常被使用,所以我们它一个"Head First Pattern荣誉奖".
有些开发人员的确是把这个编程习惯误认为是"工厂模式"(Factory Pattern).

不要因为简单工厂不是一个"真正的"模式,就忽略了它的用法.让我们看看新的比萨店类图:
